# Database Exam Tutor

Static study tool for database written exams. It presents SQL and theory exercises, keeps the database schema close at hand, and gives local rubric-style feedback while studying.

## Run locally

Open `index.html` directly in a browser, or serve the folder with any static file server.

## Test

```sh
npm test
```

## Publish

The site is intended to be published with GitHub Pages from the `main` branch.
